# 分布式超表

分布式超表是跨越多个节点的超表。通过分布式超表，您可以在多台机器上扩展数据存储，并从某些查询的并行处理中获益。

分布式超表的许多特性与标准超表的工作方式相同。要了解超表的一般工作原理，请参阅[超表][hypertables]部分。
